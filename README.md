# Repositório Institucional
Aplicação Web de um repositório institucional da FAETERJ-Paracambi feito com MySQL, Express, Vue.js e Node.js.
Seria o MEVN Stack só que no lugar do MongoDB foi utilizado MySQL
# Estrutura
``` bash
├───root
    ├───backend
    │   ├───db
    │   └───express
    │       └───routes
    └───frontend
        ├───public
        └───src
            ├───assets
            ├───components
            │   └───icons
            ├───router
            └───views
```
